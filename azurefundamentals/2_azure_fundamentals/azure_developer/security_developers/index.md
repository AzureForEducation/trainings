# Application Security on Microsoft Azure
Security has become “everyone’s responsibility” and as a developer you are responsible for creating secure applications in the cloud. This path will introduce you to the ways in which developing on Microsoft Azure can make your application more secure. You’ll learn exactly what is your responsibility and what Azure will do for you.
